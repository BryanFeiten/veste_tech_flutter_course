## Como executar o conteúdo de aula
No dart, o entrypoint da aplicação é o código descrito na função `main`, mas como temos vários arquivos em aula, precisamos apontar para qual arquivo queremos rodar. 

Com isso, o comando terá a seguinte estrutura:
- `dart run` + `{CAMINHO_DO_SEU_ARQUIVO}`

Exemplo:
`dart run ./lendo_o_terminal.dart`
ou
`dart run ./2_concatenacao.dart`
